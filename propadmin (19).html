⚙ Configurar Gmail
Cómo obtener el Client ID (5 min):
1. console.cloud.google.com → Nuevo proyecto → nombre: PropAdmin
2. APIs & Services → Enable APIs → Gmail API → Activar
3. Credentials → + Create Credentials → OAuth 2.0 Client ID
4. Si pide pantalla de consentimiento → External → solo nombre y email
5. Application type: Web application
6. Authorized JavaScript origins: null
7. Copia el Client ID y pégalo abajo
